# Lab 08: Abstract Classes

## Objectives
- Understand abstract classes and abstract methods
- Practice implementing required methods in subclasses
- Work with geometric shape calculations

## Files
- `Shape.java` - Abstract parent class
- `Circle.java` - Circle implementation
- `Rectangle.java` - Rectangle implementation
- `Triangle.java` - Triangle implementation

## Key Concepts
- Abstract classes with `abstract` keyword
- Abstract method implementation
- Area and perimeter calculations
