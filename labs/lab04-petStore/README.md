# Lab 04: Pet Store

## Objectives
- Practice class inheritance with `extends`
- Learn to use constructors with `super()`
- Understand method overriding

## Files
- `Pets.java` - Parent class for all pets
- `Dog.java` - Dog subclass implementation
- `Cat.java` - Cat subclass implementation
- `Hamster.java` - Hamster subclass implementation

## Key Concepts
- Inheritance
- Constructor chaining
- Method overriding with `@Override`
