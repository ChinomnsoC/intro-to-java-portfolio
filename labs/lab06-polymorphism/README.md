# Lab 06: Polymorphism

## Objectives
- Understand polymorphism and class hierarchies
- Practice using the `instanceof` operator
- Work with ArrayLists of parent types

## Files
- `Moving.java` - Main lab implementation
- `Bird.java`, `Parrot.java`, `Toucan.java` - Bird class hierarchy
- `Feline.java`, `Lion.java`, `Tiger.java` - Feline class hierarchy
- `Reptile.java`, `Lizard.java`, `Gecko.java` - Reptile class hierarchy
- `Animals.txt` - Animal data file

## Key Concepts
- Polymorphism
- Type checking with `instanceof`
- Class hierarchies and inheritance
