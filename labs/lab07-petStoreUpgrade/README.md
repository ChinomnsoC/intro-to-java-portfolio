# Lab 07: Pet Store Upgrade

## Objectives
- Practice method overloading
- Understand multiple method signatures with same name
- Build upon inheritance concepts from Lab 04

## Files
- `Pets.java` - Parent class for all pets
- `Dog.java` - Dog subclass with overloaded methods
- `Cat.java` - Cat subclass with overloaded methods
- `Hamster.java` - Hamster subclass with overloaded methods
- `Bird.java` - Bird subclass with overloaded methods

## Key Concepts
- Method overloading
- Different parameter types and counts
- Method signatures
