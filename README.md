# CS162: Introduction to Java

Coursework from Colorado State University's CS162 course (Asynchronous, Fall 2025)

## About

First formal Computer Science course taken as a software engineer with 5 years of Python experience but no academic CS background. Focused on learning Java fundamentals, OOP principles, and software engineering best practices.

## Course Topics

- Object-Oriented Programming (Inheritance, Polymorphism, Encapsulation)
- Java Type System and Generics
- File I/O and Exception Handling
- Design Patterns (MVC, Factory, Strategy)
- Data Structures and Algorithms
- Testing and Debugging

## Repository Structure

- **labs/** - 10 weekly lab assignments
- **exams/** - 2 coding exams (midterm and final)
- **project/** - Final project: Text-based RPG game

## Key Highlights

- **Knight's Game Project**: Full text-based RPG with combat system, character management, and persistent save/load functionality
- Technologies: Java, CSV parsing, OOP design patterns
- Grade: B (86%)
